# Digital Planner & Journal

## Project Overview
**Digital Planner & Journal** is a web application that helps users manage their journals, tasks, and notes efficiently. It provides authentication, CRUD operations for journals, and email notifications for important events. This repository includes both frontend and backend details.

---

## Tech Stack

### Frontend
- **React.js** – For building the user interface  
- **Tailwind CSS** – Styling framework  
- **React Router** – Navigation  
- **Axios** – HTTP client for API requests  

### Backend
- **Node.js** – JavaScript runtime  
- **Express.js** – Backend framework  
- **Supabase** – Database & authentication  
- **Nodemailer / Brevo** – Email notifications  
- **Render / Vercel** – Deployment  

---

## Features
- User authentication (Sign Up / Login)  
- Create, read, update, and delete journal entries  
- Upload and store profile images  
- Email notifications for events (if implemented)  
- Responsive design
